Klene

Safe cleanup for Arch Linux. Klene gives you a clean desktop app and a fast CLI for reviewing cleanup opportunities before anything gets removed.

View on GitHub Preview first GUI and CLI

Why I made Klene

I wanted an Arch Linux cleanup tool that felt clear and trustworthy. Too many cleanup tips jump straight to commands that are easy to misunderstand. Klene slows that down in the right way. It scans first, explains what it found, and keeps real cleanup behind confirmation.

What it cleans

Pacman cache

Remove old cached package files while keeping recent versions.

Orphan packages

Review packages that are no longer required by anything else.

System journal

Trim old journal logs to save space without guessing.

User cache

Target low-risk cache folders instead of wiping everything under your home directory.

Trash and thumbnails

Clear the easy, low-risk clutter that builds up over time.

AUR and Flatpak

Review common extra package caches when those tools are installed.

Safety-first design

  • Preview is always non-destructive.
  • Real cleanup requires confirmation.
  • Package removal gets extra care.
  • Protected paths are refused.
  • The same backend powers both the GUI and CLI.

GUI and CLI

Klene is built for both kinds of users. If you want a polished desktop utility, the PySide6 GUI gives you a clear dashboard, safe previews, and branded desktop integration. If you prefer the terminal, the Typer CLI gives you scan, doctor, about, and cleanup commands with the same safety model.

Who it is for

Klene is for Arch Linux users who want to stay tidy without falling back to risky one-liners. It is especially useful if you want a cleanup tool that explains itself, shows its work, and stays conservative by default.
